🔕 Suppression rules

Mute future alerts that match a pattern (signature, category, src/dst IP, or max severity). Matched alerts still appear in the dashboard with a "muted" badge but skip Claude summarization and Discord notification. Use an optional TTL for time-limited mutes (e.g. during planned maintenance).

🔧 Tuning recommendations

SecTool scores every signature by noise vs. signal — high volume + low severity + dismissed/false-positive history make a signature a suppression candidate, while medium+ severity or open triage keeps it safe. Each suggested rule is pinned to the observed severity ceiling, so a future escalation above that level still pages you.
